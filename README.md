# AI-Based Hate Speech Detection System

## Overview

AI-Based Hate Speech Detection System is a Machine Learning-powered web application designed to identify and classify harmful content on social media platforms such as Twitter, Instagram, Facebook, and online discussion forums.

The system analyzes user-generated text and predicts whether the content is:

* Hate Speech
* Offensive Speech
* Normal Speech

The project uses Natural Language Processing (NLP) techniques and Machine Learning algorithms to help create safer online communities by detecting toxic content before it spreads.

---

## Problem Statement

Social media platforms generate millions of comments and posts every day. Manually reviewing harmful content is difficult and time-consuming.

This project automates the detection of hate speech and offensive language using Artificial Intelligence, enabling faster moderation and improved user safety.

## How It Works

1. User enters text.
2. Text is cleaned and preprocessed using NLTK.
3. Features are extracted from the text.
4. Trained Machine Learning model analyzes the content.
5. System predicts:

* Hate Speech
* Offensive Speech
* Normal Speech

6. Prediction is displayed to the user.
